# Echo Meeting Mind

LLM-powered meeting intelligence: paste a transcript, receive structured minutes with decisions, owners, risks, and open questions. Uses **your OpenAI API key** via the in-app settings bar.

## Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · OpenAI Chat Completions (JSON mode)

## Run

```bash
npm install
npm run dev
```
